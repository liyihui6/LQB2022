## 1.十进制转其他进制

十进制转换成其他进制主要使用循环相除即可~

```java
public static String trans(int num, int base) {
    String res = "";
    int count = 0;
    while(num > 0) {
        int tmp = num % base;
        num /= base;
        res = "" + tmp + res;
    }
    return res;
}
public static void main(String args[]) {
    int num = 26, base = 4;
    string res = trans(num, base);
    System.out.println(res);
}
```
